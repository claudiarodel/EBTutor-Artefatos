\# EBTutor-Artefatos



Este repositório contém os \*\*artefatos associados ao sistema EBTutor\*\*, organizados com o objetivo de \*\*documentar os estudos de caso\*\*, \*\*disponibilizar materiais técnicos\*\* e \*\*viabilizar a reprodução parcial dos códigos e análises\*\* desenvolvidos no contexto da pesquisa.



Devido a limitações de tamanho impostas pelo GitHub, alguns artefatos — especialmente o \*\*código-fonte completo do sistema\*\* — são disponibilizados por meio de \*\*repositórios externos\*\*, devidamente referenciados.



---



\## Estrutura do Repositório



\### Estudos\_de\_Caso

Contém os \*\*estudos de caso coletados durante o desenvolvimento e os testes do EBTutor\*\*.  

\- Cada pasta corresponde a um estudo específico.  

\- Os arquivos podem estar em formatos como \*\*PDF, CSV, XLSX\*\*, entre outros, conforme a natureza do estudo.



---



\### Tecnico/Codigo\_Python\_Metodos

Contém os \*\*códigos em Python utilizados nos métodos técnicos\*\* associados ao EBTutor.  

\- Os scripts estão organizados de forma a \*\*facilitar a compreensão e a reprodução das análises\*\* realizadas.



---



\### Tecnico/Banco\_de\_Dados

Contém o \*\*esquema do banco de dados\*\* utilizado pelo EBTutor (aproximadamente \*\*130 KB\*\*), suficiente para fins de \*\*documentação e reprodutibilidade experimental\*\*.



---



\### Tecnico/Codigo\_EBTutor

Devido ao seu tamanho, o \*\*código-fonte completo do sistema EBTutor\*\* não está armazenado diretamente neste repositório.  

O código foi compactado e disponibilizado em um repositório externo, acessível por meio do link informado no arquivo contido nesta pasta.



---



\## Download e visualização dos arquivos



Alguns arquivos disponibilizados neste repositório — em especial \*\*planilhas no formato `.xlsx`\*\* — \*\*não são visualizados corretamente pela interface do GitHub\*\*. Para acessá-los de forma adequada, é necessário realizar o \*\*download dos arquivos\*\* e abri-los localmente no computador.



\### Como realizar o download



\- Para baixar \*\*um arquivo individual\*\*, clique sobre o nome do arquivo e utilize a opção \*\*Download\*\*.

\- Para baixar \*\*todo o conteúdo do repositório\*\*, clique no botão \*\*Code\*\* e selecione a opção \*\*Download ZIP\*\*.



Após o download, os arquivos podem ser abertos normalmente em softwares compatíveis, como \*\*Microsoft Excel\*\* ou \*\*LibreOffice\*\*.
